# supplies-api

This is a simple REST API using Spring Boot, Hibernate/JPA and H2 written in Kotlin.

Versions:
* Kotlin - 1.3.21
* Spring Boot - 2.1.6.RELEASE
* Gradle - 5.4.1

to build:
```bash
gradle clean build
```

to run:
```bash
gradle bootRun
```
